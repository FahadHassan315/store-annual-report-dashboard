# Store Annual Report Dashboard

This project analyzes annual store performance using Google Sheets and Power BI-style visuals.

## Overview
The dashboard provides insights on:
- Order status and fulfillment performance  
- Sales by gender and region  
- Channel contributions across platforms (Amazon, Myntra, Flipkart, etc.)  
- Monthly order vs. sales trends  

## Tools Used
- Google Sheets (dashboard & data visualization)  
- Power Query & Pivot Tables  
- ReportLab (for PDF report generation)
